# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-06

### Added

- Initial release
- Python type hints best practices (Python 3.10+)
- Pydantic V2 data modeling patterns
- Code style guidelines (logging, path handling, hashing)
- Code quality tools configuration (Ruff, mypy, pytest)
- Best practices for dependency management and project organization
- Multi-platform support:
  - Claude Code
  - OpenCode
  - Codex CLI
  - Cursor
  - Gemini CLI
  - GitHub Copilot CLI
  - Factory Droid
  - Roo Code
  - TRAE
- CLI tool for easy installation
- npm package for distribution
